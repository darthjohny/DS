# Архив исследовательских документов

Здесь лежат вторичные исследовательские документы по редким и пограничным
случаям.

Они сохраняются как:

- доказательная база для ВКР;
- история научных гипотез и их проверки;
- источник для возможного возврата к редким пограничным случаям.

Эти документы не относятся к текущему активному рабочему контуру,
но остаются воспроизводимой частью исследования.
