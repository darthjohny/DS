# Архив Исследовательских Документов

Здесь лежат вторичные deep-dive документы по исследовательским расследованиям.

Они сохраняются как:

- доказательная база для ВКР;
- история научных гипотез и их проверки;
- источник для возможного возврата к редким edge-cases.

Эти документы не относятся к текущему active operational contour,
но остаются воспроизводимой частью исследования.
