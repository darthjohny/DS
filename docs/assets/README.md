# Docs Assets

Дата актуализации: 13 марта 2026 года

## Назначение

Каталог хранит изображения, на которые есть прямые ссылки из
документации и notebook-ов.

## Канонические файлы

- `gaia_archive_crossmatch_ui.png`
- `gaia_archive_validation_ui.png`

Эти файлы используются в:

- `docs/preprocessing_pipeline_ru.md`
- `notebooks/eda/00_data_extraction_and_preprocessing.ipynb`

## Что не нужно коммитить

Одноразовые локальные screenshots с именами вида
`Снимок экрана *.png` не считаются каноническими doc-asset-ами и
игнорируются через `.gitignore`.

Если для документации нужен новый screenshot, его стоит:

1. переименовать в осмысленное имя;
2. сослаться на него из конкретного `.md` или `.ipynb`;
3. только после этого оставлять в репозитории.
