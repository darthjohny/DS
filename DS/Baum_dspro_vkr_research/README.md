# Baum_dspro_vkr_research

Исследовательский проект ВКР.

Цель:  
Построение статистической модели сходства звезд на основе гауссовых распределений
для задач приоритизации наблюдений.

---

## 1. Архитектура

Проект состоит из трех основных модулей:

src/
│
├── eda.py
│   Подготовка данных:
│   - фильтрация карликов
│   - формирование признаков
│   - контроль качества
│
├── model_gaussian.py
│   Статистическая модель:
│   - глобальная z-нормализация
│   - оценка параметров по классам
│   - Mahalanobis distance
│   - similarity score
│   - сохранение/загрузка параметров (JSON)
│
└── db_test.py
    Проверка подключения к PostgreSQL
    и доступности ключевой вьюхи пайплайна

---

## 2. Используемая модель

Для каждого спектрального класса вычисляется:

- вектор средних
- ковариационная матрица
- детерминант
- число обусловленности (cond)

Расстояние:
Mahalanobis distance

Similarity:
1 / (1 + d)

Все расчеты ведутся в нормализованном (z-score) пространстве.

---

## 3. Подключение к БД

Подключение осуществляется через SQLAlchemy.

Рекомендуемый способ — через переменную окружения:

DATABASE_URL=postgresql+psycopg2://USER:PASSWORD@HOST:PORT/DB

Если переменная не задана — используются параметры по умолчанию
(локальное подключение).

Файл:
src/db_test.py

---

## 4. Запуск

Обучение модели:

python src/model_gaussian.py

Проверка БД:

python src/db_test.py

---

## 5. Зависимости

Указаны в requirements.txt

Создание окружения:

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
