# Архив Исследовательских Тестов

Этот каталог не входит в активный `pytest`-контур проекта.

Здесь лежат исследовательские проверки, которые были полезны на этапе
диагностики и deep-dive анализа, но больше не считаются рабочим слоем
регулярного качества.

Правила для этого каталога:

- файлы не обязаны соответствовать текущему стандарту активного test-слоя;
- архив не используется как аргумент в пользу рабочего покрытия проекта;
- при необходимости повторного использования проверки сначала переносятся
  обратно в активный контур `tests/unit/...` и только после этого
  приводятся к полному инженерному стандарту.
