# Логи калибровки decision_layer

В этой папке хранится история ручной калибровки итогового `final_score`.

Формат файлов:
- `iteration_001.md`
- `iteration_002.md`
- `iteration_003.md`

Назначение:
- фиксировать параметры одной итерации;
- не терять контекст между прогонами;
- быстро сравнивать гипотезы и результаты;
- сохранять ход работы для ВКР и последующей отладки.

Новый файл создаётся командой:

```bash
python src/decision_layer_logbook.py
```

После запуска появится следующий по номеру markdown-файл с готовым шаблоном.
