# Логи работы программы

В этой папке хранится журнал прогонов основного пайплайна.

Формат файлов:
- `run_001.md`
- `run_002.md`
- `run_003.md`

Назначение:
- фиксировать каждый запуск оркестратора;
- сохранять источник данных, параметры и итоговые наблюдения;
- не терять контекст между прогонами;
- быстро восстанавливать, что именно запускалось и какой результат это дало.

Новый файл создаётся командой:

```bash
python src/program_run_logbook.py
```

После запуска появится следующий по номеру markdown-файл с готовым шаблоном.
