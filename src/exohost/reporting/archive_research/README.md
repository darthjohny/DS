# Архив исследовательских reporting-модулей

Здесь лежат вторичные обзорные модули, которые больше не входят в активный
рабочий контур проекта.

Эти файлы сохраняются как:

- воспроизводимый след исследовательских расследований;
- база для возможного возврата к редким кейсам;
- поддержка архивных notebook и архивной документации.

Они не считаются частью активного рабочего слоя и не должны
использоваться активным кодом без явного возврата из архива.

Этот каталог сохраняется как внутренний архив и не должен использоваться как
часть активного reporting-контура без явного возврата в рабочий слой.
