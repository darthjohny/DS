# Архив исследовательских модулей данных

Здесь лежат вспомогательные модули загрузки данных для архивных
исследовательских веток.

Эти файлы:

- не входят в активный контур работы с данными;
- не считаются частью текущего рабочего пайплайна;
- сохраняются только как воспроизводимая поддержка архивных notebook и
  архивных документов.

При возврате такого файла в активную работу его сначала нужно вернуть в
рабочий слой и привести к текущему инженерному стандарту.

Этот каталог сохраняется как внутренний архив и не считается частью активного
контура работы с данными проекта.
